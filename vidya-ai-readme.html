§ 02 — Core Innovations

Not a chatbot.
A learning companion.

Vidya acts as a Socratic "Didi" (big sister) — guiding, celebrating, and adapting to every child's unique journey.

🧠
Proactive Agentic Empathy
Vidya autonomously executes background actions using Gemma 4's function-calling capabilities. She detects when you succeed, awards XP, fires confetti, and unlocks badges — all without you asking. From Starter to Scholar to Legend.
→ detect_success()
✓ award_xp(amount=150)
★ unlock_badge("Problem Solver")
🎉 trigger_confetti()
// All running on-device, offline
🃏
Smart Flashcards
When Vidya notices you struggling, she proactively saves the concept to your personal flashcard deck for later practice — automatically.
📖
Vocabulary Builder
Difficult words encountered during storytelling are automatically saved to your personal dictionary. Learn while you read.
🎨
Handwriting Canvas
A multi-color digital scratchpad for visually working through equations or sketching ideas. Math is tactile — Vidya knows this.
🫀
Mood Tracker
Emotional wellbeing is deeply tied to learning. An emoji-based daily journal lets children express themselves, helping Vidya adjust her tone and approach.
⏱
Focus Timer
A built-in Pomodoro timer to help distracted students build lifelong healthy study habits — one 25-minute session at a time.

§ 03 — How It Works

Gemma 4's natural language output is wired directly to on-device Kotlin functions via a custom Android Intent bridge — making the model a first-class citizen of the app's runtime.

§ 06 — Get Started

Up and running
in minutes.

Designed for hackathon reviewers and educators alike. No cloud keys, no API costs, no setup headaches.

1
Clone the repository
Get the source code onto your machine.
git clone https://github.com/your-repo/vidya-ai
2
Open in Android Studio
Open the Android/ folder as a project and let Gradle sync completely.
3
Connect device & run
Connect an Android 12+ device or start an emulator, then click Run ▶
4
One-time model download
On first launch, Vidya downloads the Gemma 4 model. After that — fully offline forever.
